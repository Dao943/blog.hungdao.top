Visual Studio Code，微软官方编辑器，主要记录安装过程与使用中的问题，如何解决问题的方法

简介

Visual Studio Code（简称 VS Code ），由 Microsoft（微软）提供，运行于 Mac OS X、Windows 和 Linux 之上的，针对于编写现代 Web 和云应用的跨平台源代码编辑器，可在桌面上运行，并且可用于 Windows，macOS 和 Linux。它具有对JavaScript，TypeScript和Node.js的内置支持，并具有丰富的其他语言（例如 C++，C＃，Java，Python，PHP，Go）和运行时（例如 .NET和 Unity）扩展的生态系统

  • 官网

解决下载速度慢

将下载链接 https:// 与 /stable 中间字段，替换为：vscode.cdn.azure.cn

示例如下：

  • 替换前：https://az764295.vo.msecnd.net/stable/97dec172d3256f8ca4bfb2143f3f76b503ca0534/VSCodeUserSetup-x64-1.74.3.exe

  • 替换后：https://vscode.cdn.azure.cn/stable/97dec172d3256f8ca4bfb2143f3f76b503ca0534/VSCodeUserSetup-x64-1.74.3.exe
